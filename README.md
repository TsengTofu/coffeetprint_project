## 專案環境設定 Project setup
- 安裝專案所需套件：``` npm install```
- 在本地端執行程式：```npm run serve```
- 在本地端打包部署：```npm run build```

## 技術 Project Tool
- VueCLI
- VueRouter
- Bootstrap 5
- Google Material Icons
